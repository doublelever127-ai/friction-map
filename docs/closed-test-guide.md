# 마찰지도 폐쇄 테스트 운영안

작성일: 2026-04-29  
대상 버전: v0.3.0 Android closed test

## 1. 목표

Google Play 폐쇄 테스트에서 마찰지도를 실제 사용자에게 14일 동안 사용해보게 하고, 첫 사용 이해도와 기록 흐름을 확인합니다.

테스트의 핵심은 기능을 많이 쓰게 하는 것이 아니라 아래 흐름이 자연스러운지 확인하는 것입니다.

```text
앱 열기 → 첫 기록 남기기 → 자주 막힌 위치 보기 → 작게 바꿔보기 카드 만들기
```

## 2. 테스터 수

새 개인 개발자 계정은 production access 신청 전 폐쇄 테스트에서 최소 12명의 테스터가 14일 동안 opt-in 상태를 유지해야 할 수 있습니다.

운영 안정성을 위해 최소 15명 이상을 모집하는 것을 권장합니다.

## 3. 테스터 모집 문구

```text
새로 만든 Android 앱 “마찰지도”의 비공개 테스트를 진행하려고 해.

마찰지도는 자꾸 미루거나 막히는 순간을 한 줄로 남기고,
반복되는 패턴을 보는 개인 기록 앱이야.

14일 동안 가볍게 써보고,
처음 봤을 때 이해되는지,
첫 기록이 쉬운지,
계속 쓸 만한지 알려주면 돼.

개인적인 내용을 꼭 적을 필요는 없고,
가벼운 예시로 테스트해도 괜찮아.
```

## 4. 14일 테스트 미션

| 기간 | 미션 |
| --- | --- |
| 1일차 | 앱 설치, 첫 기록 남기기 |
| 2~3일차 | 막힌 순간 1개씩 기록 |
| 4일차 | 자주 막힌 위치 지도 확인 |
| 5~7일차 | 작게 바꿔보기 카드 1개 만들기 |
| 8~10일차 | 카드 상태를 바꿔보기 |
| 11~13일차 | 기록을 더 남기고 지도 변화 확인 |
| 14일차 | 리뷰 설문 작성 |

## 5. 리뷰 설문 질문

1. 처음 앱을 열었을 때 무슨 앱인지 이해됐나요?
2. 첫 기록을 남기기 쉬웠나요?
3. 빠른 예시 칩이 도움이 됐나요?
4. “자주 막힌 위치” 지도는 이해됐나요?
5. 작게 바꿔보기 카드는 부담스럽지 않았나요?
6. 계속 쓴다면 어떤 상황에서 쓸 것 같나요?
7. 가장 헷갈린 단어는 무엇이었나요?
8. 불편했던 화면이나 버튼은 무엇이었나요?
9. 기록 삭제/내보내기/개인정보 안내가 신뢰감을 줬나요?
10. 공개 앱으로 올라와도 써볼 만하다고 느꼈나요?

## 6. 테스트 중 확인할 핵심 지표

- 첫 기록까지 걸린 시간
- 빈 상태에서 앱 목적을 이해했는지
- 샘플 미리보기를 실제 기록으로 오해하지 않는지
- 지도 개념을 이해했는지
- 작게 바꿔보기 카드가 숙제처럼 느껴지지 않는지
- 기록 삭제와 내보내기 위치를 찾을 수 있는지

## 7. 테스트 전 준비물

- 폐쇄 테스트용 AAB
- Play Console 앱 등록
- 테스터 이메일 목록
- 개인정보처리방침 URL
- 앱 설명과 스크린샷
- 테스트 안내문
- 리뷰 설문 링크
