# 마찰지도 Google Play 등록정보 초안

작성일: 2026-04-29  
대상 버전: v0.3.0 Android closed test

## 1. 앱 이름

```text
마찰지도
```

## 2. 짧은 설명

```text
자꾸 막히는 순간을 한 줄로 남기고, 반복되는 패턴을 보는 앱
```

## 3. 긴 설명

```text
마찰지도는 자꾸 미루거나 막히는 순간을 한 줄로 남기는 개인 기록 앱입니다.

운동하려고 했는데 또 미뤘을 때,
답장을 해야 하는데 부담스럽게 느껴질 때,
책상에 앉자마자 다른 일을 하게 될 때,
그 순간을 가볍게 기록해보세요.

기록이 쌓이면 어느 쪽 일에서, 어느 순간에 자주 막히는지 지도처럼 볼 수 있습니다.

마찰지도는 할 일을 더 늘리지 않습니다.
자주 막히는 순간을 찾아, 다음에 덜 버겁게 만드는 데 집중합니다.

주요 기능
- 막힌 순간 한 줄 기록
- 그때 느낌, 생활 영역, 막힌 위치 선택
- 자주 막힌 위치 지도
- 반복 패턴 요약
- 작게 바꿔보기 카드
- 다크모드
- 기록 내보내기
- 모든 기록 삭제

개인정보
마찰지도는 로그인 없이 사용할 수 있습니다.
사용자가 남긴 기록은 기기 안에 저장됩니다.
첫 출시 버전에서는 서버, 광고, 분석 SDK, AI API를 사용하지 않습니다.
사용자는 앱 안에서 기록을 내보내거나 삭제할 수 있습니다.
```

## 4. 카테고리 후보

1. 생산성
2. 라이프스타일

추천은 `생산성`입니다. 앱의 핵심이 치료나 상담이 아니라 개인 기록과 생활 패턴 확인에 있기 때문입니다.

## 5. 태그/키워드 후보

- 기록
- 자기이해
- 생활 패턴
- 미루기 기록
- 습관 관찰
- 개인 노트
- 오프라인 기록

## 6. 피해야 할 표현

아래 표현은 스토어 등록정보, 스크린샷, 앱 설명에서 사용하지 않습니다.

- 불안 치료
- 우울 개선
- 정신건강 치료
- 심리 진단
- ADHD 개선
- 증상 완화
- 치료 효과
- 처방
- 문제 행동 교정
- 의지 부족 극복

## 7. 스크린샷 구성안

### 스크린샷 1. 첫 기록 화면

문구:

```text
오늘 어디서 막혔나요?
자꾸 막히는 순간을 한 줄로 남겨보세요.
```

화면:

- Hero 문구
- 막힌 순간 입력 폼
- 빠른 예시 칩

### 스크린샷 2. 자주 막힌 위치

문구:

```text
기록이 쌓이면 자주 막히는 위치가 보입니다.
```

화면:

- 자주 막힌 위치 Top 3
- 마찰 지도 히트맵
- 범례

### 스크린샷 3. 작게 바꿔보기

문구:

```text
반복되는 막힘을 작은 방법으로 바꿔봅니다.
```

화면:

- 작게 바꿔보기 작성 카드
- 상태 칩
- 카드 목록

### 스크린샷 4. 내 기록 관리

문구:

```text
기록은 이 기기 안에 저장됩니다.
```

화면:

- 기록 내보내기
- 모든 기록 삭제
- 개인정보 안내

## 8. 앱 아이콘 방향

추천 방향:

- 지도 핀보다 `격자`, `점`, `작은 위치 표시` 느낌
- 의료/상담 느낌의 십자가, 하트, 병원 아이콘은 피함
- 색상은 teal, slate, white 중심
- 작은 화면에서도 알아볼 수 있는 단순한 형태

## 9. 기능 그래픽 문구 후보

```text
자꾸 막히는 순간을 한 줄로 남기는 앱
```

또는

```text
할 일을 더 늘리지 않고, 막히는 위치를 봅니다
```
