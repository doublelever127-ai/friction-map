# 마찰지도 개인정보처리방침 초안

최종 업데이트: 2026-04-29  
앱 이름: 마찰지도  
개발자 표시 이름: freemann  
문의 이메일: 출시 전 등록 예정

## 1. 문서 목적

이 문서는 Android 앱 `마찰지도`가 사용자의 기록을 어떻게 저장하고 다루는지 설명합니다.

마찰지도는 자꾸 막히는 순간을 한 줄로 남기고, 반복되는 패턴을 보는 개인 기록 앱입니다. 정신건강 치료, 심리 진단, 의료 상담, 처방을 제공하지 않습니다.

## 2. 수집하는 정보

마찰지도는 사용자가 앱 안에서 직접 입력한 내용을 기기 안에 저장합니다.

저장될 수 있는 내용:

- 막힌 순간에 대한 한 줄 기록
- 그때 느낌
- 어느 쪽 일이었는지에 대한 선택값
- 어디에서 막혔는지에 대한 선택값
- 얼마나 버거웠는지에 대한 선택값
- 작게 바꿔보기 카드에 사용자가 직접 적은 내용
- 카드 상태
- 기록 생성 및 수정 시각

## 3. 저장 위치

현재 버전의 기록은 사용자의 기기 안에 저장됩니다.

마찰지도는 첫 출시 버전에서 다음을 사용하지 않습니다.

- 서버 저장
- 로그인 계정
- 데이터베이스
- 광고 SDK
- 분석 SDK
- AI API

개발자는 사용자가 입력한 기록을 서버로 전송받거나 열람하지 않습니다.

## 4. 정보 사용 목적

앱은 사용자가 입력한 기록을 아래 목적에만 사용합니다.

- 기록 목록 표시
- 자주 막힌 위치 지도 표시
- 반복 요약 표시
- 작게 바꿔보기 카드 표시
- 사용자가 선택한 다크모드 설정 유지

기록은 사용자를 평가하거나 진단하기 위한 목적으로 사용되지 않습니다.

## 5. 제3자 제공

마찰지도는 첫 출시 버전에서 사용자의 기록을 제3자에게 제공하지 않습니다.

앱에는 광고 SDK나 분석 SDK가 포함되지 않으며, 사용자의 기록을 외부 서비스로 전송하지 않습니다.

## 6. 기록 삭제

사용자는 앱 안의 `내 기록 관리` 영역에서 모든 기록을 삭제할 수 있습니다.

삭제하면 이 기기 안에 저장된 마찰 기록과 작게 바꿔보기 카드가 사라집니다. 삭제된 기록은 앱에서 복구할 수 없습니다.

## 7. 기록 내보내기

사용자는 앱 안의 `내 기록 관리` 영역에서 자신의 기록을 JSON 파일로 내보낼 수 있습니다.

내보내기 파일은 사용자의 기기에서 생성되며, 앱 서버로 전송되지 않습니다.

## 8. 어린이와 청소년

마찰지도는 어린이를 주요 대상으로 만들지 않았습니다. Play Console 등록 시 실제 타겟 연령과 콘텐츠 등급 설문 결과에 맞춰 이 항목을 다시 확인합니다.

## 9. 향후 변경 가능성

향후 서버 저장, 로그인, 백업, 분석, AI 기능을 추가하게 되면 개인정보처리방침과 Google Play Data Safety 항목을 먼저 갱신합니다.

첫 출시 버전에서는 서버, 로그인, 광고, 분석, AI를 사용하지 않는 방향을 유지합니다.
