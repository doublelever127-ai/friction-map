# 마찰지도 v0.3 Play Console 제출 체크리스트

작성일: 2026-04-29  
대상 브랜치: `v0.3-android-playstore`  
목표: Google Play 폐쇄 테스트 제출 준비

## 1. 현재 준비 완료

- GitHub Pages 배포 워크플로 추가
- 설정 화면에 기록 내보내기 추가
- 설정 화면에 모든 기록 삭제 추가
- 설정 화면에 개인정보 안내 추가
- Android debug APK 빌드 확인
- Android release AAB 생성 확인
- Android 에뮬레이터 실행 확인
- 기록 추가, 새로고침 유지, 삭제 유지 Android WebView QA 확인


피해야 할 방향:

- 정신건강 치료 앱

```text
생산성
```

후보:

```text
라이프스타일
```

## 3-1. 앱 아이콘 준비 상태

v0.3에서 기본 Capacitor 아이콘을 마찰지도용 격자 아이콘으로 교체했습니다.

Android 앱 아이콘:

```text
android/app/src/main/res/mipmap-*/ic_launcher.png
android/app/src/main/res/mipmap-*/ic_launcher_round.png
android/app/src/main/res/drawable-v24/ic_launcher_foreground.xml
```

Play Console용 아이콘 후보:

```text
store-assets/play-store-icon-512.png
store-assets/play-store-icon-1024.png
```

제출 전 확인:

- 작은 크기에서도 격자와 카드 형태가 보이는지
- 의료/상담/진단 앱처럼 보이지 않는지
- Play Console 미리보기에서 가장자리가 어색하지 않은지

## 6. 앱 콘텐츠 항목

### 앱 액세스

로그인 없이 사용할 수 있으므로 특별한 로그인 정보가 필요하지 않습니다.

추천 답변:

```text
앱의 모든 기능은 로그인 없이 접근할 수 있습니다.
```

### 광고

현재 앱은 광고를 포함하지 않습니다.

추천 답변:

```text
광고 없음
```

### 콘텐츠 등급

앱은 개인 기록 도구입니다. 치료, 진단, 의료 조언을 제공하지 않습니다.

주의:

- 심리 진단 앱처럼 설명하지 않습니다.
- 의료 또는 건강 개선 효과를 주장하지 않습니다.

### 타겟층

첫 출시에서는 어린이를 주요 대상으로 하지 않습니다.

추천 방향:

```text
일반 성인 사용자
```

실제 Play Console 선택지는 화면 문구에 맞춰 신중히 선택합니다.

## 7. Data Safety 작성 방향

기준 문서:

```text
docs/playstore-data-safety.md
```

현재 앱 구조:

- 서버 없음
- 로그인 없음
- 광고 SDK 없음
- 분석 SDK 없음
- AI API 없음
- 사용자가 입력한 기록은 기기 안에 저장
- 사용자는 기록을 삭제하거나 내보낼 수 있음

주의:

Play Console의 실제 질문 문구에 따라 `수집` 여부 해석이 달라질 수 있습니다. 현재 설계상 개발자 서버로 전송되는 사용자 기록은 없습니다.

## 8. AAB 업로드 준비

현재 생성 확인된 파일:

```text
android/app/build/outputs/bundle/release/app-release.aab
```

단, 이 파일은 업로드 전에 서명 상태를 확인해야 합니다.

Play Console용 최종 업로드는 Android Studio에서 아래 흐름으로 만드는 것을 권장합니다.

```text
Build → Generate Signed Bundle / APK → Android App Bundle
```

서명 키는 앱 업데이트에 계속 필요하므로 안전하게 보관합니다.

## 9. 서명 키 주의

서명 키는 앱 업데이트 권한과 연결되는 중요한 파일입니다.

아직 하지 않을 것:

- 임의 비밀번호로 키스토어 만들기
- 키스토어 파일을 Git에 커밋하기
- 키 비밀번호를 문서에 적기

해야 할 것:

- 본인이 기억할 수 있는 안전한 비밀번호 사용
- 키스토어 파일을 Git 저장소 밖에 보관
- 백업 위치를 따로 마련
- Play App Signing 사용 여부 확인
