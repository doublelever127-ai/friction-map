# 마찰지도 v0.1 테스트 체크리스트

이 문서는 v0.1 기준점이 깨지지 않았는지 확인하기 위한 체크리스트입니다.

## 1. 빌드 테스트

명령:

```bash
pnpm build
```

Windows PowerShell:

```powershell
pnpm.cmd build
```

합격 기준:

- 빌드가 오류 없이 성공합니다.
- TypeScript 오류가 없습니다.
- `window is not defined`, `localStorage is not defined` 오류가 없습니다.

## 2. 기록 추가 테스트

아래 기록을 5개 입력합니다.

| 내용 | 감정 | 생활 영역 | 마찰 단계 | 강도 |
| --- | --- | --- | --- | --- |
| 운동하려고 했는데 또 미뤘다 | 부담 | 건강 | 시작 전 마찰 | 4 |
| 카톡 답장을 계속 미루고 있다 | 부담 | 관계 | 완성 마찰 | 5 |
| 책상에 앉자마자 유튜브를 봤다 | 지루함 | 디지털 | 지속 마찰 | 3 |
| 공부를 시작하려고 했는데 뭐부터 해야 할지 몰랐다 | 막막함 | 공부 | 전환 마찰 | 4 |
| 운동복을 입기 전부터 귀찮아서 멈췄다 | 귀찮음 | 건강 | 시작 전 마찰 | 2 |

합격 기준:

- 기록 카드가 5개 표시됩니다.
- 가장 나중에 입력한 기록이 위에 표시됩니다.
- 카드에 내용, 감정, 생활 영역, 마찰 단계, 강도, 작성 시간이 보입니다.

## 3. 빈 입력 방지 테스트

입력창을 비우거나 공백만 넣고 저장합니다.

```text
"     "
```

합격 기준:

- 빈 기록 카드가 생성되지 않습니다.
- 콘솔 오류가 없습니다.

## 4. 새로고침 유지 테스트

기록 5개를 입력한 뒤 브라우저를 새로고침합니다.

합격 기준:

- 기록 목록이 유지됩니다.
- 마찰 지도 개수가 유지됩니다.
- 반복 요약 값이 유지됩니다.
- 콘솔에 치명적인 오류가 없습니다.

## 5. 삭제 유지 테스트

`운동복을 입기 전부터 귀찮아서 멈췄다` 기록을 삭제합니다.

합격 기준:

- 카드 개수가 5개에서 4개로 줄어듭니다.
- `건강 x 시작 전 마찰` 개수가 2에서 1로 줄어듭니다.
- 새로고침 후에도 삭제된 기록이 다시 나타나지 않습니다.

## 6. 마찰 지도 개수 테스트

기록 5개 입력 직후 기대값:

| 위치 | 기대 개수 |
| --- | ---: |
| 건강 x 시작 전 마찰 | 2 |
| 관계 x 완성 마찰 | 1 |
| 디지털 x 지속 마찰 | 1 |
| 공부 x 전환 마찰 | 1 |
| 나머지 칸 | 0 |

합격 기준:

- 위 개수가 정확히 표시됩니다.

## 7. 반복 요약 테스트

기록 5개 입력 직후 기대값:

| 요약 | 기대 결과 |
| --- | --- |
| 자주 나타난 생활 영역 | 건강 2회 |
| 자주 나타난 마찰 단계 | 시작 전 마찰 2회 |
| 자주 함께 나타난 감정 | 부담 2회 |

합격 기준:

- 1위 항목과 개수가 맞습니다.
- 동률 항목의 순서는 달라도 괜찮습니다.

## 8. 다크모드 테스트

합격 기준:

- 다크모드 토글 버튼이 동작합니다.
- 새로고침 후에도 선택한 테마가 유지됩니다.
- 텍스트 대비가 너무 낮지 않습니다.
- 콘솔 오류가 없습니다.

## 9. v0.2 프리뷰 노출 확인

합격 기준:

- v0.1 첫 화면에 작은 실험 카드 작성 영역과 실험 카드 목록이 정식 기능처럼 노출되지 않습니다.
- 작은 실험 카드 관련 코드는 v0.2 설계 대상으로 남겨둡니다.

## 10. 모바일 QA 체크 항목

브라우저 개발자도구의 디바이스 모드에서 확인합니다.

권장 크기:

- iPhone SE
- Pixel 7
- iPad

체크 항목:

| 항목 | 기준 |
| --- | --- |
| 입력 폼 | 화면 밖으로 튀지 않음 |
| 강도 라디오 버튼 | 손가락으로 누르기 쉬움 |
| 카드 목록 | 너무 빽빽하지 않음 |
| 마찰 지도 | 가로 스크롤 가능 |
| 다크모드 | 글자가 잘 보임 |
| 삭제 버튼 | 너무 눈에 거슬리지 않음 |

## 11. 콘솔 확인

아래 오류가 없어야 합니다.

```text
Hydration failed
localStorage is not defined
Cannot read properties of undefined
Warning: Each child in a list should have a unique "key" prop
```

## v0.1 합격 판정

필수 통과 항목:

- `pnpm build` 성공
- 기록 추가 가능
- 새로고침 후 기록 유지
- 삭제 후 `localStorage` 반영
- 마찰 지도 개수 정확

위 항목이 모두 통과하면 v0.1은 합격으로 봅니다.
