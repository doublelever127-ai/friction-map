# 마찰지도 스토어 자산

이 폴더는 Google Play 등록과 검토에 사용할 그래픽 자산 초안을 보관합니다.

## 포함된 파일

- `play-store-icon-512.png`: Play Console 고해상도 앱 아이콘 후보
- `play-store-icon-1024.png`: 원본/확장용 앱 아이콘 후보

## 아이콘 방향

- 마찰 지도와 반복 위치를 떠올릴 수 있는 격자 형태
- 의료, 상담, 진단 앱처럼 보이는 상징은 사용하지 않음
- teal, white, slate 계열의 차분한 색감

## 남은 작업

- Play Console 업로드 전 작은 크기에서 식별성 확인
- 필요하면 기능 그래픽과 스크린샷용 프레임 제작
