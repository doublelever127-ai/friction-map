# AGENTS.md

이 파일은 앞으로 Codex가 이 저장소에서 작업할 때 따라야 할 규칙입니다.

## 프로젝트

- 이름: 마찰지도
- 목적: 생활 속 반복되는 마찰을 기록하고, 패턴을 발견하고, 작은 해결 실험으로 바꾸는 자기이해 도구

## 기술 스택

- Next.js App Router
- TypeScript
- Tailwind CSS
- localStorage 기반 로컬 저장
- 초기 버전에서는 서버, 데이터베이스, 로그인, AI API를 사용하지 않음

## 작업 원칙

- 한 번에 너무 많은 기능을 구현하지 않는다.
- 기존 파일을 삭제하거나 구조를 크게 바꾸기 전에 이유를 설명한다.
- TypeScript 타입을 명확하게 둔다.
- 컴포넌트를 작게 분리한다.
- localStorage 접근은 브라우저 환경에서만 하도록 안전하게 처리한다.
- 빌드 오류가 없도록 한다.
- UI 문구는 한국어로 작성한다.
- 사용자를 비난하거나 진단하는 표현을 쓰지 않는다.
- "실패", "의지 부족", "문제 있는 습관" 같은 압박적인 표현을 피한다.
- 대신 "마찰", "패턴", "가설", "작은 실험"이라는 언어를 사용한다.

## 검증 명령

- `pnpm build`

## 완료 기준

- `pnpm build`가 통과해야 한다.
- 변경한 파일 목록을 요약해야 한다.
- 다음 단계에서 이어서 작업하기 쉽게 남은 일을 정리해야 한다.
